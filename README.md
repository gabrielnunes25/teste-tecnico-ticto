# Teste Técnico

## Ferramenta

-   [Cypress](https://cypress.io/)

## Executando o projeto

Clonando projeto

```sh
git clone https://github.com/gabrielnunes25/teste-tecnico-ticto.git
```

Navegando para pasta do projeto

```sh
cd teste-tecnico-ticto
```

Instalando dependêcias

```sh
npm i
```

Rodando Cypress

```sh
npm run cy:open
```
